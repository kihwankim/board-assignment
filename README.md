# 첫 번째 인턴 과제(게시판)

## FrontEnd : Vue.js

## BackEnd : CodeIgniter4.0.4(PHP Framework)

## 페이지 구성도
* 메인 페이지(전체 게시판)
* 등록 게시물 열람 페이지
* 수정 및 생성 페이지

## 기능 요구사항
* 생성
* 삭제
* 수정

## 테이블 필수 항목
* ID(int)
* 작성자(varchar)
* 제목(varchar)
* 내용(text)
* 등록 시간(datetime)
* 업데이트시간(datetime)
